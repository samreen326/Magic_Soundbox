# Magic_Soundbox
# ✨ Arcane Spellbook

An interactive fantasy spell-casting game built with HTML, CSS, and JavaScript.

## Features

- 🔥 Cast unique magical spells
- 🎵 Sound effects for each spell
- 💙 Mana and XP system
- 🏆 Achievement notifications
- 🌙 Light/Dark theme toggle
- 📱 Responsive design

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
